# Florist Workspace UX

**Status:** Accepted  
**Tokens:** existing `tokens.css` + `@flower/ui` — no new UI library.

## Screens

- **Today** — counters → filtered lists; OrderCards; Claim next; quick actions; attention warnings
- **Work Order** — planned / availability / actual editor; sticky actions; Mark Ready review
- **Stock** — operational balances; cost hidden without permission

## Command palette

Ctrl/Cmd+K indexes the **same** nav config + permission filter as the sidebar. Not a second navigation source.

## Mobile

Sticky bottom action bar; cards not wide tables; 44×44 targets; `prefers-reduced-motion`.
